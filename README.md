# Dummy Image Product Gallery

This workspace contains a complete sample web app built from scratch with no existing data.

## Features

- Product grid with dummy products
- Dummy placeholder images from `https://via.placeholder.com`
- Search/filter by product name
- Responsive mobile-friendly layout
- Clean HTML/CSS/JS only (no frameworks)

## Run

1. Open `index.html` in your browser (double-click or `Live Server` extension)
2. Use the search input to filter products

## Replace data

- Edit `script.js` to update product entries
- Replace `https://via.placeholder.com/` URLs with real images
